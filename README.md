# Olympics Data Analysis Dashboard

An interactive web application built using **Streamlit** to analyze historical Olympics data.  
The app provides insights into medal tallies, country-wise performance, athlete statistics, and overall Olympic trends using rich visualizations.

---

## Live Demo
 https://olympics-analysis-app-ienfufjzxtlabpa5cntrs7.streamlit.app/

---

##  Features
-  Medal tally analysis (year-wise & country-wise)
-  Country-wise Olympic performance trends
-  Athlete-wise analysis (age, height, weight, medals)
-  Interactive visualizations using Plotly & Seaborn
-  Gender participation analysis over years

---

##  Tech Stack
- **Python**
- **Streamlit**
- **Pandas & NumPy**
- **Matplotlib & Seaborn**
- **Plotly**
